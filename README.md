# 📰 Blog App (Client)

A modern blog application built as part of The Odin Project curriculum.  
This project represents the **user-facing frontend** that interacts with the Blog API, allowing users to explore posts and engage through comments.

---

## 🚀 Features

- View all published posts
- Read full post content
- Comment on posts
- User authentication (login/logout)
- Responsive UI

---

## 🧱 Tech Stack

- React
- React Router
- Tailwind CSS
- Fetch API
- JWT Authentication
